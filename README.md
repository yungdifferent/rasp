# 📅 Расписание НГУЭУ — Floating Island

Веб-приложение для просмотра учебного расписания Новосибирского государственного университета экономики и управления (НГУЭУ) с плавающим док-меню в стиле «Floating Island». Данные берутся с официального сайта `rasps.nsuem.ru` и отображаются в удобном карточном формате.

## ✨ Возможности

- **Выбор подгруппы** – переключение между 1‑й и 2‑й подгруппой (сохраняется в `localStorage`).
- **Чётные / нечётные недели** – автоматический расчёт текущей недели (от 1 сентября 2024) или ручной выбор через подменю.
- **Автоматическая тема** – поддержка светлой, тёмной и системной темы (`prefers-color-scheme`).
- **Карточки дней** – каждый день недели отображается отдельной карточкой, сегодняшний день выделен.
- **Текущая пара** – подсветка пары, которая идёт в данный момент.
- **Заметки администратора** – после входа в админ‑режим можно добавлять заметки к любой паре (сохраняются локально).
- **Адаптивный дизайн** – корректно отображается на любых устройствах, включая мобильные.
- **Док с тремя кнопками**:
  - «Группа» – выбор подгруппы.
  - «Сегодня» – прокрутка к сегодняшней карточке.
  - «Ещё» – дополнительные настройки (выбор недели, смена темы, обновление, ссылка на `day.html`).

## 🚀 Демо

Вы можете открыть `index.html` в любом современном браузере. Приложение полностью клиентское, данные подгружаются с официального сервера НГУЭУ.

## 🛠 Технологии

- HTML5, CSS3 (Flexbox, Grid, backdrop‑filter)
- JavaScript (ES6+)
- Font Awesome 6
- Google Fonts (Inter)
- LocalStorage для сохранения настроек и заметок
- Fetch API для получения HTML‑страниц расписания

## ⚙️ Установка и запуск

1. Скачайте все файлы проекта в одну папку.
2. Откройте `index.html` в браузере.  
   *Некоторые браузеры могут блокировать fetch при запуске с `file://`. Рекомендуется использовать локальный сервер (например, `npx serve .` или расширение Live Server для VS Code).*

## 🔧 Настройка

Все пользовательские настройки (подгруппа, тема, выбранная неделя, заметки) автоматически сохраняются в `localStorage`.  
Чтобы сбросить настройки, откройте консоль браузера и выполните:

```js
localStorage.clear();
location.reload();
```

## 👑 Режим администратора (заметки)

1. Кликните 5 раз по футеру «NSUEM Schedule • Island …» внизу страницы.
2. Введите логин `ydkova` и пароль `okeyshawty`.
3. Теперь при клике на любую пару можно добавить или изменить заметку.
4. Заметки сохраняются в `localStorage` и видны только вам.

## 🧩 Расширение функционала

Вы можете добавить новые страницы (например, `live.html`, `notes.html`), используя те же стили и парсер расписания. Ключевые функции (`ScheduleParser.parseSemester`, `normalizeTime`, `addMinutes`) расположены в глобальном объекте `ScheduleApp` – их легко переиспользовать.
